# Required Sources for Country Scores

This is the pruned source set needed before populating `data/country-scores.ts`.
It keeps DOI/title pairs and unavoidable no-DOI datasets that provide country-level scores, rankings, or strong country-ordering anchors for the eight Culture Map scales.

Use direct score tables first, ordinal country evidence second, and broad proxies only when direct sources are missing.

## Sources Not Kept for the Core Registry

The raw consensus files include many citations that are useful background but weak for score population. Do not register sources that are only Consensus export artifacts, book reviews, methodology critiques, single-context anecdotes, downstream reuse of culture scores without new country values, management-quality or inequality proxies, medical/education-only examples, or papers with no clear path to a country-level value.

Common deletion candidates across the source folders include the Consensus footer, results timelines, top-contributor tables, figure placeholders, unsupported methods counts, and references explicitly described as review-only or construct-mismatch caveats.
